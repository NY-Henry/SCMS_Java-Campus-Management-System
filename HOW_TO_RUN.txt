> WINDOWS - Command Prompt or PowerShell

i). Option 1: Using the Run Script (Easiest)


Double-click `run.bat` in the root of our project. It will compile and run automatically!

ii). Option 2: Manual Commands

```open cmd
> Navigate to project directory
eg., , cd "C:\Users\HP\Desktop\OOP Final Papeer\OOP Final"

>  Clean and create bin directory
rmdir /s /q bin
mkdir bin

> Compile all Java files
javac -d bin -cp "lib/*" src/Main.java src/database/*.java src/models/*.java src/services/*.java src/utils/*.java src/gui/*.java

> Run the application
java -cp "bin;lib/*" Main
```
