**How to run our group project

1. first import the scms_schema.sql file in any database service like WAMP, to create the database and table


2. RUN THE appalicaton (OPtions)

    i). Using the Run Script (Easiest)
            - Double-click `run.bat` in the root of our project. It will compile and run automatically!..

    ii). Option 2: Manual Commands

            ```open cmd
            > Navigate to project directory
            eg., , cd "C:\Users\HP\Desktop\OOP Final Papeer\OOP Final"

            >  Clean and create bin directory
            rmdir /s /q bin
            mkdir bin

            > Compile all Java files
            javac -d bin -cp "lib/*" src/Main.java src/database/*.java src/models/*.java src/services/*.java src/utils/*.java src/gui/*.java

            > Run the application
            java -cp "bin;lib/*" Main
            ```


    iii). Option 3: --- this is for our usecase since we're using git (bash) --- 

            ```Open bash in the current folder and run
            javac -d bin -cp "lib/*" src/**/*.java src/*.java && java -cp "bin;lib/*" Main
            ```
